Deployment checklist
  • Upload this page and all five standalone HTML pages to the repository root.
  • Keep panji_search_index.json and panji_mool_mulagram_catalog.json beside the pages.
  • Keep assets/css/videha.css in the existing assets folder.
  • Upload parsed CSV/JSONL files as archival files; the runtime pages only need the JSON search files.
