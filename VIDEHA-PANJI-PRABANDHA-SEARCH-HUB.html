📚 English Panji Books Corpus Search

Six English Panji books are searched from GitHub-hosted JSON, page by page. No PDF is needed at runtime. Search Roman/IAST forms such as Mandar, Śāṇḍilya, Halayudha, Sodarpur, Panichobh, or any English phrase.

Book corpus will load on first search.

Choose a Panji search tool

These pages are designed for GitHub Pages deployment under https://videha-ejournal.github.io/videha-sadeha/ while keeping Videha navigation under https://www.videha.co.in/.

Deployment checklist
  • Upload this page and all five standalone HTML pages to the repository root.
  • Keep panji_search_index.json, panji_mool_mulagram_catalog.json, and panji_english_books_corpus.json beside the pages.
  • Keep assets/css/videha.css in the existing assets folder.
  • Upload parsed CSV/JSONL files as archival files; the runtime pages only need the JSON search files.
