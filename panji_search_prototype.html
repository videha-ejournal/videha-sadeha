Data and deployment notes

This page reads panji_search_index.json, optional panji_search_index_addon_excel.json, and panji_mool_mulagram_catalog.json from the same GitHub Pages folder. It keeps the GitHub route while using Videha navigation links with the https://www.videha.co.in/ prefix.

  • Use Devanagari or Roman/IAST without diacritics.
  • For ambiguous tokens such as दौ / द्दौ / दु, inspect the original record before citing a final relationship.
  • Exact field matches are ranked above broad full-text matches.
