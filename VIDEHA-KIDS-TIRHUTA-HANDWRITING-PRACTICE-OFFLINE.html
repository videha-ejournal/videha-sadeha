✍️ तिरहुता Handwriting Practice

𑒞𑒱𑒩𑒯𑒳𑒞𑒰 लिपि लेखन-अभ्यास · देवनागरी ↔ तिरहुता सहायक
offline · अही पन्नापर · no upload

१. अक्षर चुनू

कार्ड पर क्लिक करैतहि तिरहुता guide canvas मे आबि जाएत। जँ glyph square box देखाइत अछि तँ सिस्टम/browser मे Tirhuta font समर्थन चाही।

२. देवनागरी सँ तिरहुता बनाउ

ई छोट converter देवनागरी अक्षर, मात्रा, अंक, अनुस्वार, चन्द्रबिन्दु, विसर्ग आ हलन्त केँ Unicode Tirhuta मे बदलैत अछि।

३. Tracing canvas

हल्का dotted Tirhuta अक्षर पर mouse/touch/stylus सँ trace करू। Esc सँ canvas साफ नहि होइत; ऊपर “Trace साफ” प्रयोग करू।

४. Printable practice lines

Print करत समय ई practice lines worksheet बनि जाएत। बच्चा पहिने guide trace करय, फेर खाली स्थानमे अपन हाथे लिखय।

तिरहुता सीखबाक स्मरण-पत्र

तिरहुता Unicode लिपि अछि। आधुनिक browser मे text रूपमे चलैत अछि, मुदा glyph देखबाक लेल सिस्टममे Tirhuta समर्थ font रहबाक चाही। PDF/PNG export canvas सँ बनैत अछि, से बच्चाक tracing सुरक्षित रहैत अछि।
