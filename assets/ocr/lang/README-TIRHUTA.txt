TIRHUTA OCR MODEL REQUIRED

No genuine Tirhuta Tesseract traineddata model is included here.

Important: Tesseract's language code 'tir' is Tigrinya, not Tirhuta. Do not rename tir.traineddata.gz to pretend it is Tirhuta; OCR output will be wrong.

For true Tirhuta OCR:
1. Train or obtain a real Tirhuta Tesseract model.
2. Save it as: tirhuta.traineddata.gz
3. Place it in: assets/ocr/lang/
4. Use OCR language code: tirhuta
